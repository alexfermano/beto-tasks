# Beto Tasks

Daily tasks picked from Beto (Bo's brain) — tracked by Alex.

Each morning, Claude scans Beto, Bo's calendar, Slack #exec-ops, and Gmail to surface a 360 view of what's in play. Alex picks the tasks to own, and they get logged here as dated markdown files.

## Structure

```
tasks/
  2026-05-25.md
  2026-05-26.md
  ...
```

Each file contains the tasks Alex chose that day with source context (Beto thought, calendar event, Slack thread, or email).
